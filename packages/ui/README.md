# @aksara/ui

UI components for Aksara Framework.

## Installation

```bash
npm install @aksara/ui @aksara/core
```

## Usage

### Button

```typescript
import { Button } from '@aksara/ui';

<Button variant="default" size="lg" isLoading={loading}>
  Click me
</Button>
```

### Input

```typescript
import { Input } from '@aksara/ui';

<Input
  label="Email"
  type="email"
  placeholder="Enter your email"
  error={errors.email}
/>
```

## Components

- **Button** - Versatile button component with variants and sizes
- **Input** - Form input with label and error handling
